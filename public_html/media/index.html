Data Visualisatie

Hiernaast zijn mijn autoritten met de tijd hoelang de rit duurt erbij gezet. Hier groter het verschil is tussen de oranje en blauwe balk hoe meer vertraging ik gehad heb onderweg. Ik heb ernaast een grafiek gezet van het aantal files per dag van de week. Je kan dien dat er een overeenkomst is tussen het aantal files op een dag en mijn rijstijd. Op zondag heb ik meer kilometers gereden in minder tijd dan op woensdag en donderdag omdat dit zoals op de rechter grafiek te zien is drukke dagen zijn.

Op de linker grafiek is tezien hoeveel data apps op mijn mobiel gebruiken. Hoe meer ze gebruiken hoe meer ik ze gebruik. Daarnast is een grafiek van de meest gebruikte apps op telefoons van de hele wereld. Je kan zien dat snapchat en maps twee apps zijn die ik net zoals de rest van de wereld veel gebruik. Maar wat opvallend is is dat ik wel veel whatsapp gebruik maar de rest van de wereld niet.

User Experience

Met mijn proftaak groepje hebben we een scenario, storyboard en prototype gemaakt voor ons proftaak project: AstroBlaster.

De ervaring die gebruikers met het programma van ons product zullen hebben is redelijk simpel, de gebruiker begint op het start scherm waar hij/zij kan kiezen om een spel te starten of om te kijken naar de highscores. Wanneer de gebruiker op start klikt kan er gekozen worden uit verschillende spel modus. Na het kiezen van een spel modus wordt het spel gestart. Wanneer het spel is afgerond kan de gebruiker zijn/haar naam intoetsen wordt de score opgeslagen. Dan wordt de gebruiker terug gestuurd naar het hoofdmenu waar hij/zij op highscores kan klikken om doorgestuurd te worden naar het highscore scherm.
