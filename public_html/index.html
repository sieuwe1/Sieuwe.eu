CortanaRGB

CortanaRGB is an complete RGB controller for multiple RGB accessories. With cortanaRGB you can control RGB accessories with your voice and use build in effects. For cortanaRGB you only need: a Arduino, a windows 10 PC, 3 n-channel mosfet, and common annode RGB accessories. Check out the build tutorial and a demo with the link below.
